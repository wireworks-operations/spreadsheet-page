# Interactive Excel Material Design

An interactive HTML tool styled with **Material Design 3** for editing Excel-like tables directly in your browser.

## Features
- Material Design 3 UI with pillbox buttons and icon-only toolbar
- Dark mode toggle
- Collapsible horizontal toolbar
- Color picker and preset swatches
- Export with colors preserved
- Batch edit actions: add/delete rows & columns, merge cells
- Create/delete sheets dynamically

## Setup
1. Download the ZIP file.
2. Extract the contents to your desired folder.
3. Open `index.html` in any modern web browser.
